IA na prática: Acelerando o desenvolvimento e garantindo a qualidade com um fluxo de trabalho automatizado por IA

![Node.js CI](https://github.com/matthewOG06/ia-dev-workflow/actions/workflows/ci.yml/badge.svg)

 - Contexto

A empresa simulada enfrenta dificuldades para entregar novas funcionalidades com rapidez sem perder qualidade. Quando o foco é acelerar as entregas, a quantidade de bugs aumenta. Quando o foco é qualidade e testes, os prazos acabam não sendo cumpridos.

Esse cenário ficou mais evidente no último trimestre, quando a equipe passou a trabalhar sob maior pressão devido a uma campanha de marketing.

 - Objetivo

Demonstrar, de forma prática, como o uso de ferramentas de Inteligência Artificial pode ajudar no equilíbrio entre velocidade e qualidade no desenvolvimento de software.

 - Uso da IA no Projeto

O GitHub Copilot foi utilizado como apoio na criação da função principal do projeto e também na escrita dos testes automatizados, reduzindo o tempo gasto com código repetitivo.

O GitHub Actions foi utilizado para automatizar a execução dos testes a cada push no repositório, garantindo um feedback rápido sobre possíveis erros.

 - Estrutura do Projeto
   
src/        -> código principal da aplicação  
tests/      -> testes automatizados  
.github/    -> workflow de CI com GitHub Actions  

 - Tecnologias Utilizadas

Node.js

Jest

GitHub Copilot

GitHub Actions

 - Conclusão

Com a adoção de um fluxo de trabalho assistido por IA, foi possível melhorar a produtividade da equipe e aumentar a confiabilidade do código entregue. O uso combinado do Copilot e do GitHub Actions mostrou-se eficiente para reduzir erros e acelerar o desenvolvimento.
